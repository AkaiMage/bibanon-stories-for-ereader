AppleMailBomber is a simple mail bomber for Mac OS X. It uses AppleScript to quickly send emails to a given target. It uses Apple's Mail.app to send the mail. You need to have an account configured for use in Mail.app to use AppleMailBomber. It is recommended to use a fake Gmail account.
